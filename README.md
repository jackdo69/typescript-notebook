# TypeScript Notebook

A personal notebook for learning and exploring TypeScript concepts and libraries.

## Purpose

This repository serves two main purposes:

### 1. `/concepts`
A collection of basic TypeScript concepts learned while using TypeScript in everyday development. This directory contains:
- Core TypeScript features and syntax
- Type system fundamentals
- Advanced patterns and techniques
- Best practices and common pitfalls

### 2. `/libraries`
Examples and demonstrations of TypeScript/JavaScript libraries for everyday tasks. This directory showcases:
- Practical library implementations
- Integration examples
- Usage patterns
- Real-world scenarios

## Getting Started

### Prerequisites
- Node.js (v14 or higher recommended)
- npm or yarn

### Installation

```bash
npm install
```

### Building

Compile TypeScript files:
```bash
npm run build
```

Watch mode for development:
```bash
npm run build:watch
```

## Project Structure

```
typescript-notebook/
├── concepts/           # TypeScript concepts and patterns
├── libraries/          # Library examples and demos
├── dist/              # Compiled JavaScript output
├── tsconfig.json      # TypeScript configuration
└── package.json       # Project dependencies
```

## TypeScript Configuration

The project uses strict TypeScript settings with:
- Target: ES2020
- Module: CommonJS
- Strict mode enabled
- Source maps and declarations generated
